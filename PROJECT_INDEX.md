# Project Index

This repository contains the GitHub-facing record of a completed computational genomics discovery campaign. The files are arranged so the default repository directory shows the results first, with the deeper report and candidate archives available for audit.

## Main Files

| File | What It Contains |
|---|---|
| `README.md` | Plain-language summary of what the project found and why the candidates matter. |
| `00_screening_framework.md` | Explanation of the shared screening logic used across the completed modules. |
| `00_data_sources_and_dependencies.md` | Source and tool backbone behind the campaign. |

## Report Package

The detailed report and candidate archives are preserved under:

- `outputs/perusing_biological_datasets_report_package_2026-05-18/Perusing_Biological_Datasets_Research_Report_Detailed.md`
- `outputs/perusing_biological_datasets_report_package_2026-05-18/pdf_build/Perusing_Biological_Datasets_Research_Report_Detailed.pdf`
- `outputs/perusing_biological_datasets_report_package_2026-05-18/candidate_archives/`

The archive contains 187 packetized candidate records plus the 4,071-row extremophile enzyme atlas index. These are computational candidates for wet-lab follow-up, not validated biological products.

## What Was Intentionally Left Out

The uncompleted original AMR-surveillance and enzyme-pathway-anomaly experiment files are not part of this GitHub package. Large downloaded databases, raw execution intermediates, model caches, and temporary Kaggle outputs are also excluded.
